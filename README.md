# Secure-Notes-App

Secure your Notes by using this application with 4 digit number.

### Built the application using these
  - Kotlin
  - MVVM
  - Room 
  - LiveData
  - ViewModel
  - RecylcerView for animations on data changes
  - Animations on List and Activity Transitions
